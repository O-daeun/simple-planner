# 할 일 목록

## 캘린더 기능 개선

### [ ] 캘린더 캡션에 연도/월 선택 드롭다운 추가

- **파일**: `components/ui/calendar.tsx`
- **설명**: 캘린더의 월/년도 표시 부분(예: "2026년 1월")을 클릭하면 연도와 월을 각각 선택할 수 있는 드롭다운이 나타나도록 구현
- **현재 상태**: 현재는 화살표(`<`, `>`)로만 월 이동 가능
- **목표**:
  - 캡션 클릭 시 연도 선택 드롭다운과 월 선택 드롭다운 표시
  - 드롭다운에서 선택한 연도/월로 빠르게 이동 가능
  - `react-day-picker`의 `captionLayout` 옵션 활용 또는 커스텀 컴포넌트 구현
